# bot-template
This script creates a blank template for developing an aiogram bot

@ Example_Bot
|_ .env
|_ .env.example
|_ .gitignore
|_ main.py
|_ @ config_data
|  |_ config.py
|_ @ filters
|  |_ filters.py
|_ @ handlers
|  |_ other.py
|  |_ user.py
|_ @ lexicon
|  |_ lexicon.py
|_ @ middlewares
    |_ inner.py
    |_ outer.py


* create a folder Example_Bot
* download the script *into Example_Bot*
* give it execution rights
* run the script *in Example_Bot*
